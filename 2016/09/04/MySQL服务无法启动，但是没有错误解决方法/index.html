记：安装MySQL日志

今天忽然想学习MySQL的使用方法，因此心血来潮去MySQL官网上下载了MySQL 5.7.14版本的压缩包，本来按照步骤解压后配置了环境变量，添加了“my.ini”文件在文件中书写了自己的安装路径，然后启动之后，却出现了<安装及服务无法启动，服务没有报告任何错误>,于是自己上网查了好多发现，是因为自己安装的解压包中没有Data包只有先生成了这个包之后才能启动成功，安装Data包的方法是在你的安装目录下运行“mysqld –initialize”语句生成Data包然后”mysqld install“安装以后启动MySQl服务输入命令：net start mysql
最后如愿成功启动！！！！嘻嘻，偶尔写写自己的学习经历还是不错的。。。。
